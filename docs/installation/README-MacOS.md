# Petición de Boletines (Instrucciones para macOS)

Este proyecto te permitirá generar solicitudes de boletines. Sigue los pasos a continuación para configurar y ejecutar el proyecto en un entorno macOS.

## Instrucciones para preparar el archivo

1. Descarga el archivo `.txt` con el código del proyecto.
2. Ábrelo con un editor de texto como **TextEdit**, **Sublime Text**, **VS Code** o **Nano** en la terminal.
3. Guarda el archivo con el nombre **`Petición de Boletines.py`**. Asegúrate de que esté guardado con la extensión `.py` y no como archivo de texto.

## Ejecución del proyecto

### Requisitos previos

Debes tener Python instalado en tu sistema. macOS suele traer Python 2.x preinstalado, pero para este proyecto es preferible que uses Python 3. Puedes verificar si tienes Python 3 instalado ejecutando:

python3 --version

## Pasos para ejecutar en macOS

1.Abre la aplicación Terminal.

2.Verifica si tienes Python instalado escribiendo el siguiente comando:

	python3 --version

	-Si está instalado, verás la versión de Python.
	-Si no lo está, sigue las instrucciones anteriores para instalar Python 3 usando Homebrew.

3.Navega hasta la carpeta donde guardaste el archivo Petición de Boletines.py con el comando cd:
 	Por ejemplo, si guardaste el archivo en la carpeta Documentos, usa:

	cd ~/Documents
4.Dale permisos de ejecución al archivo (opcional):

	chmod +x Petición de Boletines.py

5.Ejecuta el archivo con el siguiente comando:
	python3 Petición de Boletines.py
